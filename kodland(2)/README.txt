Инстурменты для реализации:
-python
-telebot
-sqlite3
-кружечка кофе
Умения:
1. Бот  поддерживает интерактивные элементы - кнопки, для удобного взаимодействия со студентами.
2. Студенты имеют возможность запросить расписание всех уроков школы, нажав на соответствующую кнопку.
3. Разработчику предоставлина документацию по использованию бота для администраторов онлайн-школы и их студентов.
4. Бот предоставляет простой и интуитивно понятный интерфейс для студентов.
